# 🖨️ NSE Prints – Printing & Xerox Services Website

A modern, high-performance website built for **NSE – Non Stop Enterprises**, a printing and stationery service provider located in Mumbai.  
The platform allows customers to explore services, contact the shop instantly, and use an **AI Print Assistant** to estimate printing requirements.

The website is designed with a **modern Full-Stack architecture** to ensure high performance, beautiful UI, and excellent user experience across all devices.

---

# 🚀 Features

- ⚡ Ultra-fast website powered by **Next.js**
- 🎨 Modern **dark-mode UI with glassmorphism design**
- 🤖 **AI Print Assistant** to help users estimate printing needs
- 📱 Fully **responsive design** (mobile, tablet, desktop)
- 💬 **WhatsApp integration** for instant customer contact
- 📝 Contact form that sends messages directly to email
- ✨ Smooth animations and interactive UI
- 📊 Built-in analytics for monitoring performance
- 🔍 SEO optimized for better Google visibility

---

# 🧑‍💻 Tech Stack

The NSE Prints website uses a **modern industry-standard Full-Stack architecture**.

---

## 🎨 Frontend (UI & User Experience)

- **Next.js 15 (React 19)**  
  Handles the Single Page Application (SPA), fast page loading, and SEO optimization.

- **Tailwind CSS 4**  
  Utility-first CSS framework used for responsive layouts, dark mode styling, and modern UI design.

- **Framer Motion 12**  
  Used to create smooth animations, parallax effects, and interactive UI transitions.

- **Shadcn UI + Radix UI**  
  Provides accessible, professional UI components such as navigation menus, forms, and modals.

- **Lucide React**  
  Modern vector icons used across the website.

---

## ⚙️ Backend (Logic & Infrastructure)

Instead of a traditional backend server, the website uses **Next.js built-in server features**.

- **Next.js Server Actions**  
  Handles contact form processing and AI assistant logic securely.

- **Node.js Runtime**  
  Powers the server-side environment.

- **Vercel Analytics**  
  Tracks performance metrics and user engagement.
